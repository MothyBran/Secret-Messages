❓ Häufige Fragen (FAQ)

Was ist ein Lizenz-Key?

Ein Lizenz-Key ist ein Zugangscode, der dir exklusiven Zugriff auf das Verschlüsselungstool von Secret Messages gibt. Ohne Key kein Zugriff.

Wie lange ist ein Key gültig?

Das hängt vom gewählten Paket ab: 1 Monat, 3 Monate, 12 Monate oder unlimitiert. Die Zeit läuft ab Aktivierung.

Wird der Key automatisch verlängert?

Nein! Es gibt keine automatische Verlängerung oder Abo-Fallen. Du kaufst einmalig und behältst volle Kontrolle.

Was passiert, wenn ich mein Gerät verliere?

Die verschlüsselten Nachrichten bleiben geschützt. Ohne deinen Code kann niemand sie lesen.

Kann ich einen Key verschenken?

Ja, das ist möglich. Jeder Key ist einmalig aktivierbar. Ideal für Freunde oder Partner.
